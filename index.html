InfDB

Informatik Databasklass

Usage: Download InfDB.jar and jaybird-full-2.2.5.jar and import them as libraries to your project. Import the database class and exception;

import oru.inf.InfDB; import oru.inf.InfException;

Create a field using the database class example: private InfDB db;

Use the javadoc in javadoc.zip for reference to how the methods work.
